Front Page Announcements are only for accounts with a BLOG. If the client has a blog we can create a space on the 'front page' to give them special notice for any one post. This allows them to temporarily draw attention to that post. Examples: a New Release , or a special deal.

Plug-in Needed:
Advanced Custom Fields
- import json file to ACF

Category:
- create category "Front Page Announcement"
- take note of the category id

Add to Front Page:

<!-- IF HAS BLOG - FRONT PAGE ANNOUNCEMENT  -->
<?php
    $homepageAnnouncement = new WP_Query(array (
    'posts_per_page' => 1,
    'post_type' => 'post',
            'category_name' => 'front-page-announcement'
    
    )); 

    if ($homepageAnnouncement->have_posts()){ ?>
<section id="fp-announcement">
    <?php
        while($homepageAnnouncement->have_posts()) {
        $homepageAnnouncement->the_post(); ?>
    <div class="fp-announcements container">
        <h2 class="about-starter-hdr"><?php the_field('fp_headline'); ?></h2>
        <div id="tra-img">
            <img  src="<?php the_field('fp_image'); ?>">
        </div>
        <p><?php the_field('fp_supporting_text'); ?></p>
        <a class="catch-announce " href="<?php the_permalink(); ?>">See Details</a>
        <?php
        }?>
    </div>
</section>
<?php
    }
    wp_reset_postdata();?> 



Remove 'category' from Sidebar:
You don't want Front Page Announcement to show up in the sidebar as a clickable category. These are meant for temporary category placement.

<div class="pt-5">
    <h2 class="cat-title">Categories</h2>
    <?php  
        wp_list_categories( array (
			'show_count' => 1,
			'style' => "",
			'title_li' => "",
			'exclude' => '7',  //'7' being the id assigned to new category "Front Page Announcement"
		));
    ?>
</div>
